## 文章结构模板（七段式）

每篇 KL 遵循以下骨架：

```
H. 标题 + 导读
   ├── 标题：主体 +「动作」+ 结果
   └── 【养虾快讯】导读（≤3句，≥3个数字）

①  爆点开场（1-3段）
    └── 最炸裂的数据/事实/冲突

②  事件主线（3-6段）
    └── 按时间线/因果链叙述核心事件

③  人物/案例特写（2-4段）
    └── 当事人故事 / JS 实践佐证

④  技术深潜（核心干货，编号列表）
    └── N 条技术要点，每条一句话标题 + 通俗解释

⑤  社区反应 / 延伸发现（2-4段）
    └── 业界大佬点评、社区二次创作、衍生项目

⑥  原因/争议/背景补充（2-3段）
    └── 追问"为什么"，提供深层背景

⑦  收尾金句 + 前瞻（1-2段）
    └── 一句话定性事件意义 + 后续关注点
```
