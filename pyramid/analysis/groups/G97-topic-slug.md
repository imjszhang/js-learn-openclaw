Atoms 按"写作规约与避坑指南 -> 核心反转与主题定义 -> 演变阶段与实证数据 -> 结构设计与修辞策略"的逻辑排列。先明确什么不能写（EO-01/02, ET-01/02/03），再确立核心论点（EO-03/08/11, ET-05/06），接着展开具体的演变轨迹和 Agent 角色变化作为证据（EO-04/05/06/07, ET-07/11/12），最后落实到文章结构、开头结尾及标题的具体执行策略（EO-09/12, ET-09/10/14/15/18/19）。
