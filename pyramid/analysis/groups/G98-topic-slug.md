# G98: 模型同质化时代，真正的竞争优势在于利用模型构建的系统而非模型本身的智商

> 当顶级模型成为巨头专属或行业标配时，普通人应停止追逐模型参数，转而通过 Harness Engineering 构建不依赖旗舰模型的自有系统以确立壁垒。
