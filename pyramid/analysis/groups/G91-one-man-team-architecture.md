# G91: OpenClaw 的本质不是单一工具，而是一个可被雇佣的“最小化团队”

> 引入 OpenClaw 的目的不是让人“一个人干三个人的活”，而是利用杠杆效应让人从执行者升级为指挥者，通过情报、资料、开发等虚拟角色的分工协作实现效能跃迁。
