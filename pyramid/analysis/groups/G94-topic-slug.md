# G94: AI 内容创作 Skill 必须采用"代表作投喂 - 人工改写 - 差异分析"的迭代闭环，并坚守"人定方向判活人感，AI 做执行填肉"的分工底线

> 构建高可用写作 Skill 的核心不在于一次性完美 Prompt，而在于通过 3-4 轮"AI 试写 - 人工重写 - 差异复盘"的迭代循环校准风格，且必须将 L4"活人感"终审与核心价值判断牢牢掌握在人类手中。

## 组内逻辑顺序

遵循"Skill 构建流程 -> 迭代策略与误区 -> 四层自检体系详解 -> 人机分工边界 -> 核心价值哲学"的逻辑顺序。首先阐述如何从零构建 Skill (KS-01~05)，接着明确迭代轮次限制与预期管理 (KS-06~08)，随后深入解析四层自检的具体标准 (KS-09~14)，进而划定人与 AI 在创作各环节的具体分工 (KS-15~19, KS-22, KS-25)，最后升华至对"真实人感"与情感价值的哲学判断 (KS-20~21, KS-23~24)。
